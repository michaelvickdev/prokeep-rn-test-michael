# react-native-reqres.in

## Stack
- React Navigation
- Axios
- Formik
- React Query
- Expo
- Jest

## Installation

On the command prompt,run the following commands:

```sh
npm install
```
  or
```sh
yarn
```


## Running on your devices
  * Run the following commands at your favourite terminal:
	* For expo `npm run dev` 
	* For ios `npm run ios` 
	* And android `npm run android`

## Test

On the command prompt,run the following commands:

```sh
npm run test
```
